# leptos-daisyui-rs を簡単に利用できるツール

現状このコンポーネントを crate をインストールして、ドキュメントを読んで import して使うようにしているが、正直やりづらい。
なので、CLI も新しく提供して、セットアップ、ミニマムインストールなどをできるようにするツールを開発したい。

## やりたいこと

`init` すると以下が行われる(mono でも workspace でも動くように)

- `rust` のプロジェクトであるかどうか
  - じゃなければ標準エラーに書き込んで終了
- `leptos` が dependencies に含まれているか
  - ユーザーに選択させて、最新のバージョンを追加してあげる
- `trunk` がユーザーのコマンドにインストールされているか
  - ユーザーに選択させて、最新のバージョンをインストールしてあげる
- `tailwindcss` のプロジェクトであるか
  - もしそうじゃないならセットアップしてあげるか聞く
- コンポーネントの管理
  - 基本的に、プロジェクトのメインクレートの `src/generated/*` 配下にこちらで定義したコンポーネントを書き込んであげて、そのファイルがあるかどうかでコンポーネントの有無を判定することとする。

`list` でコンポーネント一覧を見れる

- install 可能なコンポーネント一覧を見たい（かつそれがインストー済みであるかも表示）
- installed なコンポーネント一覧を見たい

`add` すると、コンポーネント等を `src/generated/*` 配下に追加する。あとは、generated.rs を作ってあげて、そこに、import したものを列挙して使えるようにしてあげる必要もある。
かつ tailwindcss の input.css 的なものに該当のコンポーネントの CSS を追加する。
css については、多分高銀ツールで作成したことがわかる何かしらマークできるように、しておくのがいいと思う。

基本的コンポーネントはユーザー編集可能なものであるとしておき、もし上書きして初期化したい場合の option を追加しておいて、基本上書きせず、存在することをユーザーに伝える。

そのほか便利なものがあれば追加したい。
